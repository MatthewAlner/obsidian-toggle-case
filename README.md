# Toggle Case

This [Obsidian](https://obsidian.md) plugin adds a command to toggle the case of your selection between `lowercase` => `Title Case` => `UPPERCASE` 

| Command                                        | Shortcut \* |
|------------------------------------------------|-------------|
| Toggle Case                                    | Not set     |
